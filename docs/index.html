A decentralized P2P encrypted mesh network.
The more nodes — the faster and stronger for everyone.

⚡ Run a node — one command

curl -sSL https://raw.githubusercontent.com/Rockenrol2017/swarm/main/install.sh | bash

Linux · Any VPS · Root access · Port 7437/UDP open · ~50 MB RAM

Why S.W.A.R.M.?

🔒

End-to-end encrypted

ChaCha20-Poly1305 + X25519 key exchange. Every packet is authenticated and encrypted.

⚡

QUIC transport

Modern UDP-based protocol. Fast handshake, multiplexed streams, no head-of-line blocking.

🌍

Decentralized

No central servers. Every node you run makes the network stronger and faster for everyone.

🏠

Whole-network protection

Set up once on your home server. Every device — phone, TV, console — is automatically protected.

📡

Satellite optimized

Works on high-latency satellite links. BBR congestion control, 30s handshake timeout.

🔓

Open source

GPL v3. Full source code on GitHub. Verify every line yourself.

How it works

[Phone, laptop, TV, console]
        ↓ transparent proxy — no config needed
[S.W.A.R.M. client node — your home server]
        ↓ QUIC · ChaCha20-Poly1305 · X25519
[Bootstrap node — VPS in another country]
        ↓
[Internet]
